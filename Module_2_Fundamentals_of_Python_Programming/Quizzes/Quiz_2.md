### Quiz Questions ###

1. **QUESTION 1**: 
   How do you define a function in Python?
   - A) `def function_name():`
   - B) `function function_name()`
   - C) `define function_name():`
   - D) `function_name def():`
   - **Correct Answer**: A) `def function_name():`

2. **QUESTION 2**: 
   What keyword is used to define a function in Python?
   - A) `function`
   - B) `def`
   - C) `define`
   - D) `func`
   - **Correct Answer**: B) `def`

3. **QUESTION 3**: 
   Which of the following is the correct way to define a function that takes no parameters and prints "Hello, World!"?
   - A) 
     ```python
     def hello():
         print("Hello, World!")
     ```
   - B) 
     ```python
     define hello():
         print("Hello, World!")
     ```
   - C) 
     ```python
     function hello():
         print("Hello, World!")
     ```
   - D) 
     ```python
     func hello():
         print("Hello, World!")
     ```
   - **Correct Answer**: A) 
     ```python
     def hello():
         print("Hello, World!")
     ```
